# GNG 发行核心原则（更新于 2026-04-13）

## 产品基本信息
- **产品名**：GNG（Gold and Glory）
- **品类**：中世纪地牢 Extraction RPG（近战搜打撤）— 不是 Extraction Shooter
- **单局时长**：10-20分钟（不以时长作为差异化卖点，相对MOBA稍短）
- **当前阶段**：测试期（4月关服维护 → 7月大版本重新开服）
- **评分**：Google Play 3.7（紧急触发线3.8，目标4.0+）

## 7月版本关键信息
- **3C升级**：核心体验大改，迎合MOBA爽感（技能释放体验类似MOBA）
- **Arena竞技场**：新PVP天梯玩法
- **PVE模式**：降低新手门槛
- **前几次测试留存未达标** → 7月版本是3C大改后的重新验证
- **MOBA TA假设需要重新验证**（3C变化可能改变最佳TA定义）

## 发行阶段核心原则

1. **留存是唯一目标** — 一切决策以留存为准
2. **评分紧急预案触发线**：低于3.8启动预案（暂停OM + 差评分析 + 紧急修复）
3. **D2及格线50%（必须大于50%），D7目标25%、及格线20%**
4. **ingame痛点改动 >> 外围活动/营销**
5. **发行活动只辅助触达核心用户，不承担留存责任**
6. **世界杯方案轻量**：只是沾边，非重点投入

## 各市场定位
- **ID/VN**：老市场版本更新，有老用户池可做回流
- **MX/CO**：南北全面开放
- **AR/CL**：新开放市场，冷启动
- **BR**：全新市场（停服太久无老用户基础），按新游戏发行处理
- **全区7月同步上线**：目的是拿各区对比数据

## 竞品
- 唯一端游参考：Dark and Darker（Ironmace，Krafton持有手游授权）
- 手游近战搜打撤赛道：目前无直接竞品（蓝海）
- 手游竞品（非直接）：Arena Breakout(FPS)、Delta Force(FPS)、MISFITZ(Top-down)
- **最大威胁**：DaD手游版上线

## 7月测试核心任务
1. 验证3C改版后的留存是否达标
2. 重新验证TA定义（MOBA用户是否仍是最佳，需分层campaign验证）
3. 拿到各区对比数据（ID/VN/MX/BR/AR/CL横向对比）
4. 评分管理（开服后目标4.0+）

## 素材优先级
1. **3C对比（第一）** — 直接展示手感质变，MOBA用户最在意
2. **Arena实况（第二）** — 竞技吸引力
3. **PVE教学（第三）** — 新手友好

## 不做的事
- 不使用KOL（本次版本更新）
- 不做BP等商业化宣传
- 不做大型世界杯联动（无预算）
- 巴西不打情怀召回牌（无老用户）

## 已确认的决策
- KOC为主（Star Cultivation Program）
- ASA小预算测试
- 拉美TA以MOBA渗透为锚点
- 外挂：后置检测处理，公开Ban名单
