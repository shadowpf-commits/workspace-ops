# GNG 发行核心原则

## 产品基本信息
- **产品名**：GNG（Gold and Glory）
- **品类**：中世纪地牢 Extraction Shooter，近战搜打撤
- **测试阶段**：留存测试阶段

## 发行阶段核心原则

1. **留存是最优先目标** — 一切决策以留存为准
2. **评分紧急预案触发线**：低于 3.8 分启动紧急预案
3. **D7 是 Check Point** — 是否放量不重要，数据验证为主
4. **世界杯方案轻量**：无预算，纯 ingame event + 少量 OM 素材
5. **发行活动的定位**：只是辅助产品触达核心用户 + 有机会传播，不承担留存责任
6. **外围改动 vs ingame 痛点**：外围改动/活动都不如 ingame 痛点改动有效

## 越南暑假非世界杯国家策略

越南暑假（6-8月）与世界杯重叠期：
- 本地化创意素材方向待确认
- 可结合 Arena 挑战赛做本地赛事

## 外挂处理策略
- 后置检测 + 处理为主，非前置防止
- 参考 Valorant/Apex/PUBG Mobile 的公开惩罚机制
- 硬件封禁 + 统计异常检测 + 公开处刑名单

## 回流用户策略
- 100万注册用户，OM定向再营销是主力
- KOC策略：真实回流内容 > 大KOL
- Arena挑战赛/Discord老玩家邀请赛

## 留存调研方法
- 参考 retention-and-engagement skill（29位专家框架）
- 重点：找到 "Aha Moment" 和 "Magic Number"
- D7 是分水岭，超过后留存稳定
- 调研对象：测试期D7流失用户 + 活跃用户

## 评分目标
- 当前：3.7
- 目标：7月前 4.0+
- 紧急预案触发：低于 3.8

## 已确认的决策
- 不做 BP 等商业化宣传（测试期重点是留存）
- KOL 不大量使用，以 KOC 有机爆发为主
- ASA = Apple Search Ads，测试期小预算
- 拉美 TA 以 MOBA 渗透为锚点（而非 Free Fire）
