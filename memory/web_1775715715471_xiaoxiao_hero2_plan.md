## 一、产品分析

### 1.1 产品定位

| 维度 | 描述 |
|------|------|
| 主品类 | Casual / Puzzle |
| 子类型 | Match-3 + Battle RPG（消消乐战斗混合） |
| 核心循环 | 消除→积累能量/道具→释放技能/战斗→通关 |
| 商业化 | 预计：体力系统 + 抽英雄 + Battle Pass |
| 平台 | Mobile（iOS + Android） |

**产品核心逻辑：**  
消消英雄2 = Royal Match（消除玩法骨架）+ 英雄技能组合 + PVE战斗推进

### 1.2 竞品分析

#### "大成"竞品（年营收>$50M 或下载量>3M）

| 竞品 | 厂商 | 年营收估算 | 核心机制 | 主要市场 |
|------|------|-----------|---------|---------|
| Royal Match | Dream Ott | ~$100M+ | 纯消除 + 剧情推进 | 全球 |
| Gardenscapes | Playrix | ~$200M+ | 消除 + 模拟经营 | 全球 |
| Homescapes | Playrix | ~$150M+ | 消除 + 模拟经营 | 全球 |

#### 消消英雄2 vs Royal Match 优缺点对比

| 维度 | Royal Match | 消消英雄2（预估） |
|------|------------|-----------------|
| **玩法深度** | 中等（纯消除） | ✅ 战斗系统提供策略深度 |
| **角色养成** | 弱（装饰为主） | ✅ 英雄技能组合是核心 |
| **上手门槛** | 低（纯休闲） | ⚠️ 略高（需理解战斗机制） |
| **长线留存** | 中等（靠关卡稀缺性） | ✅ 英雄升级/技能树可支撑 |
| **付费设计** | 体力 + 道具 | 体力 + 抽卡 + 英雄养成 |
| **美术风格** | 明亮卡通，成熟 | 待确认 |
| **社交/公会** | 弱 | 待确认 |

#### 市场空白识别

| 空白 | 描述 | 发行机会 |
|------|------|---------|
| **品类空白** | Match-3 + 真正战斗RPG结合的产品极少 | 直接进入 |
| **体验空白** | 现有消消乐产品缺乏策略深度和养成目标 | 微创新切入 |
| **用户空白** | 喜欢消消乐但觉得"太简单没追求"的用户 | 定位"有深度的休闲" |

### 1.3 发行推导

**差异化核心：** "消消乐的外壳 + RPG的灵魂"  
不是纯消消乐，而是**消消乐战斗RPG**——让休闲玩家也能体验到"英雄养成 + 技能组合"的成就感。

**目标用户画像：**
- 核心用户：Royal Match / Candy Crush 重度玩家（寻求新刺激）
- 延伸用户：三消玩家中喜欢RPG/卡牌养成的群体
- 潜在用户：轻度RPG玩家想找"更简单"的选择

**上手节奏建议：**
- 测试期：聚焦"上手门槛"验证，确保休闲用户能理解战斗逻辑
- 爆发期：选在目标市场休闲游戏高峰期（如暑期）

## 六、发行时间线

### Phase 1：预热期（上线前4-8周）

| 时间 | 动作 |
|------|------|
| 第1-2周 | 建立官方社媒账号（TikTok/YouTube/Discord） |
| 第3-4周 | 发布游戏预告片，开始KOL铺垫 |
| 第5-6周 | 开启预约活动（双平台预约奖励） |
| 第7-8周 | 封闭测试（KOL+社区志愿者），收集留存数据 |

**里程碑：** 预约量达到50,000+（ID+MX合计）

### Phase 2：测试期（上线后第1-4周）

| 时间 | 动作 |
|------|------|
| 第1周 | 软上线（ID/MX），观察自然增长 |
| 第2周 | 开始UA投放（小额测试），优化素材 |
| 第3周 | 分析R2/R7，调整目标用户定向 |
| 第4周 | 决定是否进入Phase 3爆发期 |

**里程碑：** R7 > 8%，CPR < $0.60

### Phase 3：爆发期（上线后第5-12周）

| 时间 | 动作 |
|------|------|
| 第5-6周 | 扩大UA投放（BR + US），增加KOL投入 |
| 第7-8周 | 配合市场热点（如World Cup预热、暑期） |
| 第9-12周 | 全面铺开，冲击榜单 |

**里程碑：** 月自然下载 > 100,000，月流水 > $50,000

### Phase 4：长线运营（第3个月起）

- 每月大版本更新 + 节日活动
- Battle Pass赛季制
- 新英雄/新技能持续推出
- 社群活动常态化

## 七、风险与建议

### 7.1 主要风险

| 风险 | 等级 | 应对方案 |
|------|------|---------|
| **上手门槛过高** | ⚠️ 中 | 测试期重点监控"第3关留存"，必要时简化战斗教学 |
| **消消乐玩家不接受战斗系统** | ⚠️ 中 | 强调"消消乐原汁原味，战斗只是bonus" |
| **竞品 Royal Match 挤压** | 🔴 高 | 差异化传播，KOL强调"新鲜感" |
| **ID市场宗教敏感期（Ramadan）** | ⚠️ 中 | 提前2周完成测试，避开斋月前两周 |
| **TikTok渠道纯付费效果差** | ⚠️ 中 | 始终搭配KOL，TikTok有机内容优先 |

### 7.2 关键建议

1. **测试期聚焦ID+MX两个市场**，成本低、反馈快，便于快速迭代
2. **前两周重点盯R2**（次日留存），如果R2<30%说明上手体验有严重问题
3. **素材方向：** 休闲玩家对"英雄收集+技能展示"敏感度高于纯消除演示
4. **ASA优先于FB：** 在US/UK市场，Apple Search Ads的CPR和R7明显优于FB
5. **留存测试阶段不要急于买量**，先确保产品留存达标再进入正式UA
