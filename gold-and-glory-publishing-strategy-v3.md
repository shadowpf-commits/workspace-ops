## Section 1: 产品基本介绍

### 1.1 产品一句话定位

**"Quick-session medieval extraction RPG — dungeons you can finish in a coffee break."**

一款基于"精灵捕捉与养成"核心（中世纪地牢版本），融合了"轻度 MMO 社交"的高品质快节奏 Extraction RPG。

---

### 1.2 核心玩法循环

```
选择英雄 (Hero Selection)
    ↓
进入地牢 (Enter Dungeon)
    ↓
PVE打怪 + PVP抢宝 (Combat + Loot PvE monsters + Fight other players)
    ↓
提取生存/战败全失 (Extract with loot OR lose it all on death)
    ↓
资源强化英雄 (Upgrade hero with extracted resources)
    ↓
Ranked / Guild 循环 (Climb ranks, guild battles)
```

---

### 1.3 产品基础数据

| Dimension | Detail |
|-----------|--------|
| **Primary Genre** | RPG |
| **Hybrid Genres** | Battle Royale + Extraction Shooter + Hero-based MOBA |
| **Monetization** | F2P with IAP + Seasonal Battle Pass |
| **Session Length** | ~5 minutes (dungeon raid) |
| **File Size** | ~840 MB (optimized for low-end devices) |
| **Platform** | Android (Google Play primary); iOS likely planned |
| **Developer** | Studio Dois Private Limited |
| **Current Rating** | 3.7 (Google Play) — ⚠️ 低于健康线 4.0 |

---

### 1.4 竞品比较

**直接参考（玩法最接近）：**

| 游戏 | 开发商 | 特色 | 评分 | 与 G&G 差异 |
|------|--------|------|------|------------|
| **The Elder Scrolls: Blades** | Bethesda | 地牢RPG，无PVP extraction | 4.0 | 题材相似但无PVP extraction |
| **Naruto: Slugfest** | 腾讯 | MMO地牢 + PVP | 3.8 | 依赖IP，战斗系统不同 |
| **NEW STATE Mobile** | PUBG Studios | BR + 抽卡英雄 | 3.9 | 题材现代军事，非中世纪 |
| **Squad Busters** | Supercell | 休闲MOBA BR | 4.1 | 无英雄收集，无地牢 |

**G&G 的真正蓝海：** 中世纪题材 + 地牢 Extraction + 快节奏（5分钟）+ 手游端，**目前无强力直接竞品**。

---

### 1.5 优劣势分析

| 优势 | 劣势 |
|------|------|
| 差异化明确：5分钟地牢extraction | 3.7分偏低，品牌知名度不足 |
| 中世纪题材独特，PC端 Extraction 已验证 | 开发商较小，服务器稳定性问题（临时维护公告多） |
| 840MB 包体适配低端机 | 无 IP 加持，海外认知为零 |
| 赛季制 + 新英雄有运营空间 | 3.7分严重影响 ASO 和自然增长 |
| KOL 内容天然适合（5分钟片段 + PVP戏剧性）| 用户留存挑战大（需验证 5分钟机制 vs 回合制吸引力）|

## Section 3: 发行策略主轴

### 3.1 全球统一主轴（所有市场不变）

| # | 主轴要素 | 全球统一内容 | 为什么不变 |
|---|---------|------------|-----------|
| 1 | **核心定位** | "5分钟中世纪地牢Extraction RPG" | 差异化核心，改变即失去独特性 |
| 2 | **变现模型** | F2P + Battle Pass + 英雄直购 | 品类验证最优解，不因市场而改 |
| 3 | **内容节拍** | 每6–8周一赛季，每赛季出新英雄 | 运营节奏和社区期待建立在稳定节奏上 |
| 4 | **核心叙事** | "5分钟 → 地牢 → 战斗 → 提取 → 爽" | 转化率最高的叙事方式，跨文化有效 |
| 5 | **赛季KPI框架** | D1 > 30%，D7 > 15%，CPA < $15 | 全球统一的健康度基准 |
| 6 | **Battle Pass结构** | 50级结构，$9.99美国基准 | 结构统一，参数（价格/奖励）按区域调整 |
| 7 | **评分改善目标** | 3.7 → 4.0，全球同步优先级 | 评分影响 ASO，所有市场受益 |
| 8 | **KOL分级框架** | Nano/Micro/Mid/Macro 四级 | 分层逻辑通用，各市场套用 |

---

### 3.2 区域执行差异（原因驱动）

| 维度 | 差异市场 | 差异内容 | 差异原因 |
|------|---------|---------|---------|
| **定价** | 印度 / 拉美 / 东南亚 | Battle Pass 价格 $1.99–$9.99 | 购买力差异（PPP），竞争定价压力 |
| **KOL切入角度** | 每个市场不同 | 菲律宾：社交；印度：性价比；越南：攻略 | 文化偏好和用户使用场景不同 |
| **主要投放平台** | 每个市场权重不同 | 印尼 TikTok > Meta；印度 Google UAC > TikTok | 各市场平台渗透率和使用习惯不同 |
| **素材本地化深度** | 拉美 / 越南 | 本地语言 + 本地文化二创 | 文化认同影响转化率 |
| **节日营销** | 各市场不同 | 亡灵节（墨西哥）；独立日（印尼）；排灯节（印度）| 配合文化节点效果更好 |
| **活跃时段** | 各市场有时差 | 越南 UTC+7；墨西哥 UTC-6 | 投放和运营需要匹配目标用户在线时间 |

## Section 5: OM 素材框架

> ⚡ 参考 Garena 版的 L1/L2/L3 素材分层逻辑，并借鉴其"AI工具快速迭代"的思路

### 5.1 素材分层架构

```
Lv1: 创意方向（Creative Direction）
  └── 核心 Hook：Extraction紧张感 / 英雄收集 / 社交情感 / 快节奏
      ↓
Lv2: 营销卖点（Message Points）
  └── 5分钟快节奏 / 死了全没 / 英雄技能 / 组队开黑 / 抽卡不坑
      ↓
Lv3: 脚本架构（Script Structure）
  └── 具体脚本：A/B/C 版本，配合 TA 和平台定制
```

### 5.2 素材类型矩阵

| 素材类型 | 目标用户 | 沟通内容 | 素材型式 | 投放渠道 |
|---------|---------|---------|---------|---------|
| **Extraction 类** | 核心：Tarkov/DMZ/BR 玩家 | 紧张感 + 赌注 + 活着出来的爽感 | 短视频（15–30秒）| TikTok / YouTube |
| **Hero Showcase** | 次核心：MOBA/RPG 玩家 | 每个英雄独特技能 + 收集价值 | 视频（30–60秒）| TikTok / FB / YT |
| **社交类** | 泛用户：社交玩家 | 和朋友组队 + 地城探索 + 温馨感 | 短视频 / KOL实况 | TikTok / IG |
| **公平竞技类** | 竞技型玩家 | 不肝不氪 + 靠技术上分 + 天梯 | 视频 + Banner | Google UAC / FB |
| **迷因/搞笑类** | 泛用户：短视频用户 | 有趣 + 意外 + 转发冲动 | 短视频（15秒）| TikTok |
| **预约奖励类** | 所有用户 | 预约福利 + 限定奖励 | Banner + 视频 | FB / IG / YT |

### 5.3 素材 Hook 库（按情绪分类）

| 情绪 Hook | 脚本方向 | 适配市场 |
|----------|---------|---------|
| **紧张刺激** | "你敢不敢进来？进去了可能什么都带不出来" | 所有市场，Extraction 核心用户 |
| **炫耀满足** | "我从地牢带出了传说级装备，截图给你看" | 菲律宾、越南（社交炫耀文化） |
| **友情温馨** | "好朋友一起下地牢，死了也不亏" | 东南亚（强社交属性） |
| **快节奏便利** | "等电梯的时间，刚好打一把" | 印度、越南（碎片时间多） |
| **公平竞技** | "没有氪金碾压，只有技术和策略" | T1西方、印度（对付费压力敏感） |
| **收集成就** | "300种英雄，全部收集是一种什么体验？" | 所有市场（收集控） |

### 5.4 素材生产与迭代策略

**目标：** 快速测试、快速迭代，用最小成本找出最优素材组合。

**AI 辅助生产工具（参考 Garena 版）：**
- 视频剪辑：CapCut /剪映（本地化版本）
- 图片生成：Midjourney / Leonardo AI（英雄场景图）

### 7.2 KPI 预估说明

**下载预估逻辑：**
- 基础：SEA + TW MMO 市场规模 85M 下载 × 品类交叉比例
- 调整：包体（840MB）对东南亚和拉美下载有限制
- 目标转化：1–2%（基于同类手游品类基准）

**收入预估逻辑：**
- 参照：Pokemon GO、F2P 手游 RPG 品类 ARPD
- 保守：$4.8 ARPD（月均 $0.4 ARPPU × 12个月）
- 乐观：$4.8 ARPD（基于付费率提升假设）

**⚠️ 重要风险提示：**
1. **评分 3.7 是严重阻碍**：如果不改善到 4.0，实际 ARPD 可能低于预估 30–50%
2. **无 IP 加持**：和 Pokemon GO 不同，G&G 没有 IP 加持，无法依赖 IP 拉下载
3. **抄袭争议风险**：中世纪 + 抓宠 + 回合制可能在某些市场引发 Pokemon 争议
4. **拉美用户留存待验证**：测试数据显示 D7 留存 12–18%，低于品类均值，需要优化
