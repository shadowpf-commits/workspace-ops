# Gold and Glory (G&G) — 市场发行方案

## 1. 游戏品类分析

### 1.1 品类定位
- **主品类：** 竞技近战 Extraction Shooter（搜打撤）
- **Sub-type：** Hero Extraction — 有英雄系统的快节奏 Extraction，5v5 团战 + 赛季制
- **核心循环：** 进入地牢 → 搜刮 → 战斗 → 提取(或死亡)

### 1.2 核心玩法亮点
| 特色 | 说明 |
|------|------|
| 5分钟速战 | 一局5分钟，地牢→战斗→提取，高频爽感 |
| 中世纪题材 | 地牢、铠甲、剑盾、魔法，差异化明显 |
| 英雄直购 | 非抽卡，付费更透明，降低付费门槛 |
| Battle Pass | 赛季制，每赛季内容更新，保持回访 |
| 赛季制 | 创造周期性目标，维持长线活跃 |

### 1.3 商业化设计
- **Hero 直购**（主要）：直接购买英雄，无抽卡赌性
- **Battle Pass**（持续）：每赛季 $9.99，分免费/付费两档
- **皮肤/外观**（辅助）：武器、铠甲外观，不影响数值

## 3. 竞品分析

### 3.1 移动端竞品

| 竞品 | 规模 | G&G 机会 |
|------|------|---------|
| Free Fire | MAU 100M+，东南亚/拉美绝对王者 | 差异化：中世纪题材 + 精致画面 |
| PUBG Mobile | MAU 50M+ | 差异化：英雄技能 + 快节奏5分钟局 |

| Lumeshift (Moonton) | 刚软启动 | MOBA-Shooter，非纯 Extraction |

### 3.2 市场空白
**移动端 Extraction Shooter 赛道，目前无强力选手。** Free Fire 虽是射击但非 Extraction 机制。

### 3.3 SWOT

| S 优势 | W 劣势 |
|--------|--------|
| 5分钟快节奏契合移动端；中世纪题材差异化；英雄直购透明付费 | 3.7评分需尽快提升至4.0+；品类认知需从零建立；Extraction 机制有教育成本 |
| **O 机会** | **T 威胁** |
| 移动端 Extraction 空白；Free Fire 玩家可转化；Lumeshift 尚未大成 | Moonton 入局；Free Fire 推类似模式；用户对 Extraction 移动端信心不足 |

## 5. 发行策略

### 5.1 核心原则
> **OM 为主（有机内容 > 付费买量），KOL 作为 OM 放大器**
> 经验证：Flute/Tiger 等本地 meme 内容比官方宣传效果好

### 5.2 线上广告渠道优先级

| 渠道 | ID | VN | MX |
|------|----|----|-----|
| Google UAC | ✅ 主渠道 | ✅ 主渠道 | ✅ 主渠道 |
| TikTok 付费 | ⚠️ 次要 | ⚠️ 次要 | ⚠️ 次要 |
| YouTube 付费 | ✅ 辅渠道 | ✅ 辅渠道 | ✅ 辅渠道 |
| Apple Search Ads | ✅ 辅渠道 | ✅ 辅渠道 | ✅ 辅渠道 |

**CPM 参考（实际数据）：**
- YouTube 官方付费：$0.29
- TikTok 官方付费：$0.56
- UGC 活动：$2.31
- KOL 合作：$10-20

### 5.3 KOL 策略

**关键洞察（经验证）：**
- 泛游戏主播（Vtuber/General Gaming）> 垂类 Extraction 主播
- Vtuber 直播效果：DAU 平均提升 26%
- Kobo Kanaeru 直播时，60% 观众已认识游戏

**KOL 类型优先级：**
1. 🎮 泛游戏主播（General Gaming / Vtuber）- 主力
2. 🎯 FPS / 竞技类 YouTuber - 辅助
3. 🐷 本地 meme / 搞笑创作者 - 病毒传播

**KOC Star Cultivation Program：**
- 主题：**"Easy Play, Easy Win"**
- 奖励：5K Gold/视频，最多 4 视频/周期
- Top 10 获得官方推广
- 每周检查 performance metrics

### 5.4 ASO 计划

**评分修复计划（3.7 → 4.0+）：**
1. 服务器稳定性优先修复（差评主因）
2. Alpha 测试阶段主动引导评分
3. Beta 上线后每版本推送评分请求
4. 差评 24h 内官方回复

### 5.5 上线时间线（Phase 1）

```
T-4周    封闭 Alpha 测试（200-500人）
         → D2 留存 > 35%，服务器稳定
         → KOL 种子内容开始产出

T-2周    限量测试扩大（1000-3000人）
         → 开放 TikTok/YouTube KOL 内容
         → Discord 社区正式开放

T-0      公测上线（ID + VN + MX 同步）
         → Google UAC + TikTok 付费启动
         → Vtuber 直播 campaign

T+1月    数据复盘
         → R7 > 10% 达标
         → 开始 Phase 2 市场评估
```
