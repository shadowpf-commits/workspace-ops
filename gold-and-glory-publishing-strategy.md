# Gold and Glory — Publishing Strategy

> Analysis based on: Google Play Store data, official website (goldandglorymobile.com), TapTap, and competitive market research.
> Last updated: 2026-03-25

---

## 1. Game & Category Analysis

### 1.1 Category & Core Loop

| Dimension | Detail |
|-----------|--------|
| **Primary Genre** | RPG (Role Playing Game) |
| **Hybrid Genres** | Battle Royale + Extraction Shooter + Hero-based MOBA |
| **Monetization** | Free-to-play (F2P) with IAP + Seasonal Battle Pass |
| **Session Length** | ~5 minutes (dungeon raid) |
| **File Size** | ~840 MB (optimized for low-end devices) |
| **Platform** | Android (Google Play primary); iOS likely planned |
| **Developer** | Studio Dois Private Limited |

**Core Gameplay Loop:**
```
选择英雄 (Hero Selection)
    ↓
进入地牢 (Enter Dungeon)
    ↓
PVE打怪 + PVP抢宝 (Combat + Loot PvE monsters + Fight other players)
    ↓
提取生存/战败全失 (Extract with loot OR lose it all on death)
    ↓
资源强化英雄 (Upgrade hero with extracted resources)
    ↓
Ranked / Guild 循环 (Climb ranks, guild battles)
```

---

### 1.2 Unique Features

Gold and Glory occupies a specific intersection that few mobile games nail:

- **"First mobile medieval hero-based PVP dungeon exploration"** —开发者 self-description，直接点出了差异化
- **Extraction mechanic on mobile** — 这个玩法在PC (Escape from Tarkov, DMZ) 很火，手游目前很少
- **5分钟快节奏** — 对比同类地牢游戏（需要30分钟+），极大降低了门槛
- **Hero-based MOBA-like combat** — 每个英雄有独特技能，不是单纯数值碾压
- **轻度包体** — 840MB，适配低端机，受众更广
- **赛季制 + 新英雄更新** — 有内容持续更新的运营模式

**一句话定位：** "Quick-session medieval extraction RPG — dungeons you can finish in a coffee break."

### 1.3 Target Audience

**没有直接竞品 → 需要从TA入手分析：**

| 属性 | 描述 |
|------|------|
| **年龄** | 16–35岁，主力 20–28 |
| **性别** | 男性为主（70–75%），女性增长中 |
| **地域** | SEA（菲律宾/印尼/泰/越）、南亚（印度）、拉丁美洲、中东 |
| **游玩习惯** | 碎片时间、每天多次短登录；喜欢和朋友组队 |
| **支付意愿** | 中低，习惯Battle Pass；对高价英雄/皮肤有付费可能 |
| **社区属性** | Discord / TikTok活跃，18–25岁喜欢直播和KOL内容消费 |
| **文化偏好** | 中世纪/奇幻题材在SEA和南亚有稳定受众 |

**受众规模估算（可触达）：**
- SEA Mobile gamers: ~250M（2026估算）
- 其中对地牢/RPG/BR感兴趣: 约15–20% = 37–50M
- 目标转化: 0.5–1% = **185K–500K 首批核心用户**

## 2. Competitive Landscape

### 2.1 Key Competitors

**直接参考（玩法最接近）：**

| 游戏 | 开发商 | 特色 | 5M+下载? | 评分 |
|------|--------|------|---------|------|
| **The Elder Scrolls: Blades** | Bethesda | 地牢RPG，但无PVP extraction | 是 | 4.0 |
| **Siralim Ultimate** | PocketPixel | 回合制地牢，无PVP | 是 | 4.6 |
| **Naruto: Slugfest** | 腾讯 | MMO地牢 + PVP | 是 | 3.8 |
| **NEW STATE Mobile** | PUBG Studios | BR + 抽卡英雄 | 是 | 3.9 |
| **Squad Busters** | Supercell | 休闲MOBA BR | 是 | 4.1 |

**核心竞品总结：**
- **NEW STATE Mobile** — 最接近的"BR+英雄"模型，但题材是现代军事
- **Naruto: Slugfest** — 玩法最接近的日漫IP版本，但依赖IP
- **The Elder Scrolls: Blades** — 地牢+免费，但无PVP extraction

**G&G的真正蓝海：** 中世纪题材 + 地牢Extraction + 快节奏 + 手游，**目前无强力直接竞品**

---

### 2.2 SWOT Analysis

| | |
|---|---|
| **Strengths** | 差异化明确（5分钟地牢extraction）；题材独特；低端机适配；已积累5M+下载 |
| **Weaknesses** | 3.7分偏低（通常4.0+才健康）；品牌知名度不足；开发商较小；服务器稳定性问题（临时维护公告多）|
| **Opportunities** | 手游extraction品类空缺；赛季制+新英雄有运营空间；KOL内容天然适合；SEA/南亚市场快速增长 |
| **Threats** | 大厂可能复制（Supercell、腾讯）；作弊/外挂影响口碑；评分低影响ASO；用户留存挑战大 |

### 2.3 Revenue Estimate

**基础假设：**
- DAU假设：5M下载，保守估算DAU ~150–250K（3–5%活跃率）
- 付费转化：参照同类F2P mobile RPG，约1.5–3%
- ARPPU：中核RPG品类，约 $15–40/月（Battle Pass $5–10 + IAP $5–30）

**月收入估算：**

| 场景 | DAU | 转化率 | ARPPU | 月收入 |
|------|-----|--------|-------|--------|
| 悲观 | 150K | 1.5% | $15 | **$33.7K** |
| 中性 | 200K | 2% | $25 | **$100K** |
| 乐观 | 250K | 3% | $40 | **$300K** |

**年收入估算：$400K – $3.6M**

> ⚠️ 注意：以上为纯理论估算，实际收入高度依赖留存曲线和付费深度。3.7评分说明**用户体验存在明显问题**（可能是新手引导、匹配机制、或内容深度不足），如果不做改善，实际收入可能低于悲观估算。

## 3. Go-to-Market Strategy

### 3.1 Recommended Target Markets (Priority Order)

```
Tier 1 (首发):
├── 🇵🇭 菲律宾 — 地牢/中世纪题材接受度高；手游竞争相对弱；Discord/KOL文化强
├── 🇮🇩 印尼 — 2.7亿人口，最大SEA手游市场；TikTok渗透率高
├── 🇻🇳 越南 — RPG/地牢品类玩家多；付费习惯良好
└── 🇮🇳 印度 — 巨大增量市场；男性用户占主导；收入水平适合F2P

Tier 2 (6个月后):
├── 🇧🇷 巴西 — 拉丁美洲最大市场；KOL生态成熟
├── 🇹🇭 泰国 — RPG品类强；付费率较高
└── 🇹🇷 土耳其 — 中东/突厥市场跳板；用户量大

Tier 3 (视情况):
├── 🇸🇦 沙特 / 中东 — 高ARPU但获客成本高
└── 🇺🇸 / 🇬🇧 / 欧美 — 文化壁垒高，中世纪题材接受度一般
```

### 3.2 Online Advertising Plan

**平台优先级（按市场）：**

| 市场 | 主要平台 | 次要平台 |
|------|---------|---------|
| SEA / 南亚 | TikTok, Meta (FB/IG), Google UAC | Unity Ads |
| 拉丁美洲 | Meta, TikTok, Google | Unity |
| 中东 | TikTok, Meta, Google | Snapchat |
| 欧美 | Meta, TikTok, Google, Apple Search Ads | Unity, AppLovin |

**素材策略：**
- **Gameplay Trailer** — 突出5分钟快节奏 + 地牢探索 + PVP战斗
- **Hero Showcase** — 新英雄发布配合KOL推广
- **UGC / KOL二创** — TikTok短视频，15–30秒，越真实越好
- **Before/After** — 展示地牢探索前后的装备对比（extraction刺激感）

**Retargeting：**
- 下载未激活用户：7日后再触达
- 高活跃低付费用户：推 Battle Pass
- 流失用户：30天召回素材

### 3.3 KOL & Influencer Strategy

**分层KOL矩阵：**

| 层级 | 粉丝量 | 用途 | 合作形式 |
|------|--------|------|---------|
| **Macro KOL** | 100万+ | 品牌破圈 | 品牌合作视频，IGN式评测 |
| **Mid-tier** | 10万–100万 | 深度体验 | 直播试玩 + 真实反馈 |
| **Micro / Nano** | 1万–10万 | 口碑种草 | 素人真实评价，TikTok病毒内容 |
| **Content Creators** | Discord / YouTube | 长尾 | 教程、英雄攻略、赛季指南 |

**重点平台：**
- 🎵 **TikTok** — 主要内容平台；算法推荐对游戏有利
- 🎮 **YouTube** — 攻略视频、赛季内容
- 💬 **Discord** — 已有官方服务器 (discord.gg/YnMx3nwRsh)，重点建设

**G&G的天然KOL优势：** 5分钟地牢 → 短视频天然友好；PVP+extraction → 充满戏剧性内容

### 3.5 ASO Plan

**关键词策略（EN）：**
```
核心词: dungeon, RPG, multiplayer, battle royale, extraction
长尾: medieval dungeon crawler, hero shooter, team battle
情绪词: fast, quick, free, epic
```

**素材本地化（每市场）：**
- 商店图标：中世纪风格 + 角色露出
- 截图：前3张最重要，涵盖"英雄选择→地牢→战斗→战利品"全流程

**评分管理：**
- 当前3.7是**严重问题** — 需要专项改善
- 短期：引导满意用户评5星（游戏内弹窗触发）
