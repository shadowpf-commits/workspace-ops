# Gold and Glory — Publishing Strategy (V5)
> 核心信息：7月大版本更新（竞技场 + PVE + 视角变更），4–6月停服维护，10个目标市场

## 一、产品核心信息

### 1.1 产品一句话定位

**"Quick-session medieval extraction RPG — dungeons you can finish in a coffee break."**

中世纪地牢 Extraction RPG，主打 5 分钟快节奏 + 赛季制 + 英雄收集。

### 1.2 当前状态

| 维度 | 现状 |
|------|------|
| **评分** | 3.7（Google Play）— ⚠️ 低于健康线 4.0 |
| **测试状态** | 墨西哥、哥伦比亚已测试 |
| **服务器** | 4–6月停服维护中 |
| **下一次大版本** | 7月（大版本更新） |

### 1.3 7月大版本核心内容（营销噱头）

| 内容 | 玩家期待度 | 营销价值 |
|------|-----------|---------|
| **竞技场（Arena）** | ⭐⭐⭐⭐⭐ | PVP核心玩法，竞技向玩家最强噱头；可做天梯、排名的内容素材 |
| **PVE模式** | ⭐⭐⭐⭐ | 降低门槛，留存友好；适合做"新手友好"叙事 |
| **视角变更（Camera更高）** | ⭐⭐⭐ | 差异化体验；截图/视频视觉效果明显提升 |

**这三个内容是7月营销的核心素材库。**

```
Tier 1（7月全力首发）: 印尼 / 越南 / 墨西哥 / 巴西 / 哥伦比亚
```

## 七、推广节奏（以7月大版本为核心节点）

```
3月    4月    5月    6月    7月       8月     9月+
 │      │      │      │      │         │       │
 ▼      ▼      ▼      ▼      ▼         ▼       ▼
[准备]→[维护]→[维护]→[预热]→[大版本爆发]→[长尾]→[持续运营]
         无大推   无大推   KOL预告   全渠道爆发  赛季内容
                 素材储备  预约开放   Arena首发
```

### 详细阶段计划

#### Phase 1：准备期（3月–6月中）

**目标：** 评分改善 + 素材本地化 + KOL名单确认

- ✅ 评分改善专项（3.7 → 3.8）
- ✅ 所有目标市场本地化（语言、文化）
- ✅ 各国KOL名单确认（竞技/BR/Extraction类型）
- ✅ 广告素材制作（围绕7月新内容）
- ✅ 预约页面准备（Google Play + 官网）

#### Phase 2：预热期（6月中–6月底，大版本发布前）

**目标：** 制造期待感 + 预约量积累

```
6月中：
├── 官方发布7月大版本预告（竞技场 + PVE + 视角变更）
├── Nano/Micro KOL 发布"前瞻内容"（新版本爆料）
├── 预约页面正式开放
└── 应用商店截图/描述更新

6月底：
├── Mid KOL 发布7月新版本实况预览
├── TikTok 挑战赛 #GoldAndGloryArena 预热
└── Discord 社区活动（#Arena预测、福利预告）
```

#### Phase 3：爆发期（7月大版本上线）

**目标：** 最大声量冲击下载 + 榜单

```
T周（7月大版本上线）：
├── Mid/Macro KOL 同步发布Arena实况
├── TikTok #GoldAndGlory 挑战赛 + UGC激励
├── Google Play / App Store 商店专题推送
├── Battle Pass（7月赛季版）同步发售
├── 付费活动：新英雄发布 + 限定皮肤
└── 应用商店评分引导弹窗优化（情绪正向时触发）
```

#### Phase 4：长尾期（8月–）

**目标：** 留存 + 持续内容消费

- 每6–8周新赛季 + 新英雄（沿用赛季节奏）
- 持续攻略内容（竞技场战术、地牢路线）
- UGC二创官方转发激励
- 流失用户召回（30天未登录）
